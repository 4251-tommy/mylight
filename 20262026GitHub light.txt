頻閃
2017 與 LED 燈相關的光強度變化（閃爍）和諧波發射
Discuss LED 燈.
根據這份 2017 年的研究報告，關於 LED 燈的討論主要集中在其節能優勢、對電力品質（諧波與閃爍）的影響，以及這兩者之間存在的意外關聯性。
以下是根據來源整理的詳細討論：
1. 普及背景與節能優勢
全球趨勢：照明佔全球電力消耗的 19%。歐盟及加拿大、巴西、中國等國已陸續實施淘汰白熾燈的法規。
節能效益：與白熾燈相比，LED 燈可減少約 75-80% 的能源消耗。
市場現況：截至 2014 年，英國家庭安裝的燈具中有 55% 為緊湊型螢光燈 (CFL) 或 LED 燈。
2. 對電力品質的兩大影響
LED 燈的大規模引入主要影響兩個電力品質層面：諧波失真 (Harmonic Distortion) 與 光閃爍 (Light Flicker)。
諧波發射 (Harmonic Emission)：
白熾燈的電流接近正弦波，而 LED 燈由於含有電力電子元件（驅動器），會產生諧波。
研究顯示，市售 LED 燈的諧波發射差異極大，總諧波失真 (THDI) 介於 16.8% 至 189% 之間。
電力業者擔心大量安裝 LED 燈可能導致電網功率因數降低及電流諧波升高。目前對於 25W 以下的低功率 LED 燈尚無嚴格的諧波限制標準。
光閃爍感應 (Flicker Sensitivity)：
閃爍是指光亮度隨時間波動而產生的視覺不穩感，可能導致頭痛、疲勞或影響工業安全。
來源指出，LED 燈對電壓波動的反應與白熾燈截然不同。在相同電壓波動下，不同型號的 LED 燈表現出極大的差異。
頻率特性：在低頻（10 Hz 以下）時，多數 LED 燈比白熾燈更不敏感；但在高頻（20-25 Hz）時，部分 LED 燈的敏感度反而超過白熾燈。
3. 核心發現：諧波與閃爍的權衡關係
這項研究最關鍵的發現是 LED 燈的「諧波發射」與其對「電壓波動（閃爍）的敏感度」呈負相關：
低閃爍 = 高諧波：光強度變化極小（幾乎不閃爍）的 LED 燈，通常具有最高的電流諧波發射。
高閃爍 = 低諧波：對電壓波動敏感、容易產生閃爍的燈具，其諧波含量通常較低。
原因：這反映了驅動器技術的不同。為了抑制光閃爍而加裝的電子元件，往往會造成電流波形的進一步失真。
4. 其他發現與建議
與價格及功率無關：研究發現，LED 燈的價格（每瓦單價）或額定功率（瓦數）與其閃爍敏感度或諧波發射之間沒有明確關係。昂貴或高功率的燈具不一定品質更好。
標準修訂需求：現行的閃爍標準（如 IEC 61000-4-15）是基於 60W 白熾燈的特性設計的，已不再適用於評估 LED 燈。來源建議應針對高頻區域重新評估閃爍曲線。
設計兩難：若未來制定更嚴格的諧波限制，可能會導致製造商生產出對閃爍更敏感的 LED 燈。
Discuss 電力品質.
電力品質（Power Quality）在 LED 照明普及的背景下，主要聚焦於**諧波失真（Harmonic Distortion）與電壓波動導致的光閃爍（Light Flicker）**這兩大擾動現象。根據來源研究，這兩者之間存在著顯著的技術權衡關係。
以下是針對電力品質相關面向的詳細討論：
1. 諧波失真 (Harmonic Distortion)
設備特性差異：傳統白熾燈在正弦電壓下會產生接近正弦的電流，但 LED 燈包含電力電子元件（驅動器），會產生顯著的諧波。
電網衝擊隱憂：電網營運商擔憂，雖然單一 LED 燈的排放量較低，但大規模安裝後可能對電網產生負面影響，例如導致低功率因數和高水平的電流諧波。
規範現狀：目前對於 25W 以下的低主動功率 LED 燈，尚未有諧波發射的強制限制標準，功率因數亦無相關限制，但標準化組織正討論引入此類規定。
量測指標：研究透過總諧波失真（THDI）來評估，發現市售 LED 燈的 THDI 差異極大，介於 16.8% 至 189% 之間。
2. 電壓波動與光閃爍 (Voltage Fluctuations and Flicker)
定義與健康影響：閃爍是指由於光亮度隨時間波動而產生的視覺不穩感。電壓幅值的波動會導致光強度變化，可能引發頭痛、疲勞、視覺疲勞，甚至因與旋轉機械產生頻閃效應而危害工業安全。
敏感度指標（Gain Factor, GF）：研究使用「增益因子」來衡量燈具對電壓波動的敏感度。GF 越高，代表相同比例的電壓波動會產生更大的光強度變化。
標準的侷限性：現行的 IEC 閃爍計標準是基於 60W 白熾燈的特性設計的。然而，LED 燈的表現與白熾燈截然不同，且各型號間差異極大，因此現行標準可能需要針對 LED 燈重新評估。
3. 電力品質指標間的權衡 (Trade-off)
來源最核心的發現是 LED 燈的諧波發射與閃爍敏感度之間存在明確的負相關性：
高諧波發射的燈具通常具有較佳的穩定性：電流諧波失真較高（THDI > 70%）的燈具，其對電壓波動的敏感度極低（GF < 0.2），幾乎不會產生閃爍。
低諧波發射的燈具則較容易閃爍：諧波含量較低的燈具，對電壓波動的敏感度通常較高。
設計兩難：這意味著如果未來實施更嚴格的諧波排放限制，可能會意外導致市售 LED 燈變得更容易產生閃爍現象。
4. 其他潛在的干擾因素
除了常見的電壓波動，來源也提到**間諧波（Interharmonics）**以及 **2 kHz 至 150 kHz 之間的高頻擾動（Supraharmonics）**也被視為是導致 LED 燈產生閃爍的電力品質誘因。
綜合而言，討論電力品質時，不能僅獨立看諧波或閃爍，因為在驅動器設計上，改善其中一項指標（如諧波）往往會惡化另一項指標（如閃爍敏感度）。
Discuss 諧波分析.
根據來源研究，諧波分析是評估 LED 燈電力品質的核心指標之一。由於 LED 燈含有電力電子元件（驅動器），其電流波形不再像傳統白熾燈那樣接近正弦波，而是會產生顯著的諧波排放。
以下是針對來源中諧波分析的詳細討論：
1. 諧波量測與計算方法
研究團隊對 24 種市售 LED 燈進行了標準化的量測分析：
量測環境：將燈具連接至產生 230V、50Hz 純正弦電壓的程序控制電源，在燈具穩定運作數分鐘後，截取 200 毫秒 (ms) 的電流波形窗口。
分析技術：利用離散傅立葉轉換 (DFT) 計算頻譜，並依據 IEC 61000-4-7 標準將 5Hz 的頻譜值聚合成「諧波組」。
計算指標：主要指標為總諧波失真 (THDI)，計算範圍涵蓋 2 至 40 階的諧波組，並將結果表示為基波電流（第 1 階）的百分比。
2. 諧波排放的特性與發現
THDI 差異極大：測試結果顯示，不同型號 LED 燈的 THDI 介於 16.8% 至 189% 之間。其中大部分燈具 (17/24) 的 THDI 低於 70%，但有 7 種燈具超過此數值。
頻譜特徵：LED 燈的諧波主要由奇數階諧波組成，且隨著諧波階數增加，其幅值通常會逐漸降低。
與功率因數的關係：研究觀察到位移功率因數 (DPF) 與總功率因數 (TPF) 之間存在偏差，較大的偏差代表電流諧波失真較高。
3. 缺乏相關性的因素
分析發現，諧波排放的高低與以下因素沒有明確關係：
主動功率（瓦數）：無論是低瓦數還是高瓦數的 LED 燈，都可能展現高 THDI。這代表諧波失真與燈具的電力負載大小無關。
價格：較貴的燈具在諧波抑制表現上不一定優於便宜的燈具，價格並非電力品質的保證。
4. 諧波與光閃爍的權衡（關鍵發現）
諧波分析揭示了一個重要的技術權衡：高諧波排放的 LED 燈通常更不容易產生光閃爍。
低閃爍組：THDI 超過 70% 的燈具，對電壓波動極不敏感（增益因子 GF < 0.2），幾乎不會產生閃爍
